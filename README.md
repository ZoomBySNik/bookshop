## Взаимодействие с сервером
Для взаимодействия с сервером необходимо перейти на сайт [crud](https://crudcrud.com/) и получить код api

После чго укзать его в файле `./.env`
``` .env
VUE_APP_API_KEY= 'EXAMPLEKEY123'
```
